\section*{Overview}

This document summarizes the deterministic device classification used in the forecasting pipeline. Devices are labeled as:
\begin{itemize}
  \item \textbf{shiftable} if their name contains any of the following substrings (case-insensitive): \texttt{dishwasher}, \texttt{washing machine}, \texttt{washing\_machine}, \texttt{dryer}, \texttt{tumble dryer}, \texttt{wm}, \texttt{dw}, \texttt{laundry}.
  \item \textbf{non\_shiftable} otherwise.
\end{itemize}

No learning or statistical analysis is used; the rules are purely keyword-based and reproducible.
