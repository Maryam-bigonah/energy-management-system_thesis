# Real Data Sources for 24-Hour Building Energy Analysis

## Data Sources Used (All Real Data)

### 1. Building Energy Consumption
- **Source**: Building Data Genome Project 2 (BDG-2)
- **Description**: Real energy meter readings from residential buildings
- **Paper**: https://arxiv.org/abs/2006.02273
- **Data**: Hourly electricity consumption from actual buildings

### 2. Solar PV Generation
- **Source**: NREL (National Renewable Energy Laboratory)
- **Description**: Real PV generation profiles from utility-scale systems
- **Source**: NREL ATB 2024 - Utility-Scale PV-plus-Battery
- **Data**: Hourly solar generation profiles based on actual weather data

### 3. Battery Storage
- **Source**: NREL Residential Battery Storage Data
- **Description**: Real performance metrics for residential battery systems
- **Source**: NREL ATB 2024 - Residential Battery Storage
- **Data**: Actual charge/discharge efficiency and capacity data

## Building Configuration
- **Units**: 20 residential units
- **PV System**: Shared rooftop solar installation
- **Battery**: Shared battery storage system
- **Time Period**: 24-hour analysis with hourly intervals
